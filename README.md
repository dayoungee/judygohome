# judy go home
플랫포머형식의 2D게임
# 소개
주디를 공격하는 몬스터들을 피하고, 박스를 이용하여 장애물을 건너야 합니다.
몬스터를 피격하거나 음식을 먹어 점수를 획득할 수 있으며, 집으로 도착해야 게임이 종료됩니다. 스코어 4,000점 이상, 목숨 3개 유지시 퍼펙트클리어 엔딩을 확인 할 수 있습니다.
